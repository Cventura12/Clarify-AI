# Demo Flow

Objective: Showcase Interpret -> Plan -> Execute with human approval in under 3 minutes.

## Setup
- Open home page (pipeline)
- Open Integrations page (optional)
- Open AI Diagnostics page (optional)
- Prepare 2 sample inputs

## Flow (Primary)
1. Home page: paste input
   - "Follow up on my internship application and set a reminder for next week"
2. Interpret output appears
   - Highlight intent + tasks + clarification (if shown)
3. Plan output appears
   - Highlight priority score + ordered steps
4. Execute output appears
   - Show action cards
5. Approve action
   - Show draft preview + follow-up suggestion
6. Notifications + execution result

## Optional add-ons
- Go to AI Diagnostics
  - Run diagnostics on same task
  - Show reasoning steps + confidence + fallback
- Go to Integrations
  - Click Connect (mock OAuth) + Sync Calendar (mock)
