# Working Agreements

Cadence
- Daily: update tasks in `docs/plan-tracker.md` and note blockers
- Weekly: fill out `docs/weekly-update-template.md` and review progress

Definition of done
- All deliverables for the week are checked off
- Demo path works for the week
- Notes include risks and next actions

Communication
- Keep decisions in the weekly update
- Use short, explicit next steps
